Struct Documentation

template<class V, size_t DIM>
struct nv::merlin::Vector

The basic value type in Merlin-KV. Any user data should be represented as a specific type of Vector in order to be processed by Merlin_KV.

Template Parameters

  • V – type of the Vector’s elements type, which should be basic types of C++/CUDA.

  • DIM – dimension of the vector.
