enum class nv::merlin::EvictStrategy

Enumeration of the eviction strategies.

Note

The meta is introduced to define the importance of each key, the larger, the more important, the less likely they will be evicted. On kLru mode, the metas parameter of the APIs should keep nullptr, the meta for each key is assigned internally in LRU(Least Recently Used) policy. On kCustomized mode, the metas should be provided by caller.

Note

Eviction occurs automatically when a bucket is full. The keys with the minimum meta value are evicted first.

Values:

enumerator kLru

LRU mode.

enumerator kCustomized

Customized mode.
