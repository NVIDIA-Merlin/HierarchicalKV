template<class K, class V, class M, size_t D>
class nv::merlin::LocalKVFile : public BaseKVFile<K, V, M, D>

The KV file on local file system. It only save/load keys and vectors between table and file. metas are ignored in it since absolute values of metas are commonly time-variant, while the time interval between save/load calling is not deterministic, in default case. If other specified rules are required, the BaseKVFile could be inherited to implement customized read/write rules. The LocalKVFile uses compact, consecutive binary format, where keys, values, and metas are stored in seperated paths.

Template Parameters

  • K – The data type of the key.

  • V – The data type of the vector’s elements. The item data type should be a basic data type of C++/CUDA.

  • M – The data type for meta. The currently supported data type is only uint64_t.

  • D – The dimension of the vectors.

inline size_t read(size_t n, K *keys, V *vectors, M *metas) override

Read from file and fill into the keys, values, and metas buffer. When calling save/load method from table, it can assume that the received buffer of keys, vectors, and metas are automatically pre-allocated.

Parameters

  • n – The number of KV pairs expect to read. int64_t was used here to adapt to various filesytem and formats.

  • keys – The pointer to received buffer for keys.

  • vectors – The pointer to received buffer for vectors.

  • metas – The pointer to received buffer for metas.

Returns

Number of KV pairs have been successfully read.
